# PHP
## Basic notes and syntax

PHP is executed on the server, plain HTML result is sent to the browser.

Basic synatx

`<!DOCTYPE html>
<html>
<body>

<h1>My first PHP page</h1>

<?php
echo "Hello World!";
?>

</body>
</html>`

>> "Hello World"
